Two files are required:

Sample File

Information on the samples in the experiment.

Columns

This should be a tab-separated file containing the following columns:

  1. sample identifiers - The first column must be sample identifiers. They should match the sample identifiers in the Count file.

  2. condition/treatment - The condition or treatment being investigated. e.g. genotype or drug treatment

  3. stage - The developmental stage in somites

  4. sex - sex of the embryos F/M. This is optional

  5. sample_names - An optional column of samples names can be included

Count File

A file of counts for Ensembl genes for each sample

Columns

This should be a tab-separated file. A column of Ensembl Gene identifiers named Gene.ID is required. The remaining columns should be the counts for the individual samples. The column names should be of the form sample_id count

Options

Threshold for statistical significance

This is for setting the threshold for statistical significance of the adjusted p values.

The other two options panels will change once the data is loaded, based on the column names of the samples file.

Condition:

Select the column name from you samples file that describes the condition of each embryo. Currently, accepted entries in the condition column are 'wt', 'het', 'hom', 'sib' and 'mut'. The DESeq2 analysis combines wt and het entries into a sib category.

Sex:

If your samples file contains information on the sex of the embryos this can be included in the analysis to account for differences driven by differences in sex. Use the checkbox to include sex in the analysis or not. If the checkbox is ticked, please choose the correct column name from the radio buttons.

Once the options are set correctly, click "Analyse Data"

Remember, both the PCA and DESeq2 analysis will be rerun every time the "Analyse Data" button is clicked
